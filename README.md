# Sales-Prediction-Web-Application
I developed a Sales Prediction Web Application using Streamlit, designed to analyze historical sales data and predict future sales based on user inputs.

The application includes three main modules:

1. Profile Section

Displays developer information and project overview
Highlights the purpose and technologies used

2. Sales Prediction Module

Accepts inputs such as Category, Sub-Category, Segment, Ship Mode, Quantity, Discount, Year (2017–2026), and Month
Uses a Random Forest Regressor model to predict sales values
Provides model performance metrics like R² Score and MAE

3. Trend Analysis Module

Visualizes Profit vs Loss distribution
Displays Monthly Sales Trends using interactive charts
Helps in understanding business performance patterns

Technologies Used:

Python
Streamlit
Pandas & NumPy
Scikit-learn
Plotly

This project demonstrates my ability to build end-to-end data-driven applications combining data preprocessing, machine learning, and interactive visualization.
